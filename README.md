# Proyecto Web 

## 📌 Descripción
Sitio web académico desarrollado aplicando **Box Model, Flexbox y Grid** para organizar mejor las secciones y dar un estilo más atractivo.

---

## 📂 Estructura
- **index.html** → Página principal con menú y multimedia.
- **login.html** → Formulario de acceso.
- **registro.html** → Formulario de registro.
- **select.html** → Ejemplo de selectores.
- **checkbox.html** → Ejemplo de casillas de verificación.
- **fecha.html** → Selector de fecha.
- **busqueda.html** → Resultados organizados en cuadrícula.

Carpetas:
- **css/** → Archivos de estilos (`general.css`, `index.css`, `login.css`, etc.).
- **img/** → Imágenes del sitio.
- **Video/** → Videos de apoyo.

---

## 🎨 Estilos
- `general.css` → Configuración base.
- `index.css` → Menú y multimedia con Flexbox.
- `login.css` → Formulario estilizado.
- `registro.css` → Formulario con Grid.
- `select.css` → Selectores con bordes redondeados.
- `checkbox.css` → Opciones con Flexbox.
- `fecha.css` → Campo de fecha estilizado.
- `busqueda.css` → Resultados en cuadrícula con Grid.

---

## ✅ Requisitos
- Uso de **Box Model** (márgenes, padding, bordes).
- Aplicación de **Flexbox** en menús y tarjetas.
- Aplicación de **Grid** en resultados.
- Separación clara de secciones (`header`, `main`, `footer`).
- Comentarios en CSS por bloques.
